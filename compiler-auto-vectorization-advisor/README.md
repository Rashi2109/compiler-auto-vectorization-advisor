# Compiler Auto-Vectorization Advisor

An AI-assisted compiler optimization framework that:

- Detects loops
- Performs dependency analysis
- Suggests compiler optimizations
- Benchmarks GCC vs LLVM
- Uses Machine Learning recommendations
- Automates testing using GitHub Actions

## Technologies Used

- Python
- Flask
- GCC
- LLVM
- scikit-learn
- GitHub Actions

## Features

- Loop Detection
- Dependency Analysis
- SIMD Recommendation
- Benchmark Visualization
- ML Optimization Prediction
- CI/CD Automation
